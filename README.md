# 腕上验证器 (Authenticator) ⌚️

专为**非 Wear OS 安卓手表**设计的简洁、高性能、高兼容性双重身份验证 (TOTP 2FA) 应用。最低**兼容 Android 5.0 (API 21)**，让你的二次验证账号在小屏手表上更安全、更易用！

## ✨ 核心特性

- ⌚️ **手表专属排版与性能优化**：采用表盘风格（WatchFace）居中布局，针对小屏及圆形屏幕精心优化交互与内存占用，流畅不卡顿。
- ✏️ **备注与服务名称编辑**：支持点击右上角编辑按钮进入独立全屏，自由修改服务名称（如 Google、GitHub）与账号名称（如 xxx@gmail.com）。单行限制与超长省略号排版，清晰干净。
- 🔢 **超大字号验证码显示**：突出核心验证码显示，配有直观的 30 秒环形倒计时进度条，方便快速查看。
- 📷 **扫码快速添加**：集成 CameraX 与 Google ML Kit，支持直接通过手表摄像头扫描 2FA 二维码导入账号。
- ☁️ **WebDAV 备份与恢复**：支持将账号密钥安全同步备份至个人 WebDAV 服务器，更换设备或重装无忧。
- 🗑️ **长按便捷删除**：长按卡片即可触发删除确认，防止误操作。

## 🛠 技术栈

- **开发语言**：Kotlin
- **UI 框架**：Jetpack Compose & Material 3
- **本地存储**：Room (SQLite)
- **扫码框架**：CameraX + Google ML Kit Barcode Scanning
- **异步处理**：Kotlin Coroutines & Flow
- **网络同步**：OkHttp (WebDAV API)
- **最低兼容**：Android 5.0 (API Level 21)

## 🚀 构建与运行

1. 使用 Android Studio 打开本项目。
2. 完成 Gradle 依赖同步。
3. 选择连接的安卓手表或模拟器作为运行目标。
4. 点击 **Run** (`Shift + F10`) 即可安装部署。
